About Us

This is a group project by the name TechHarbor where we were striving to develop a more modern web application that allows users to buy products online that will be delivered to their address. The online application will provide a detailed overview of the products, which will be categorized to simplify the search process.

When entering the application, users with an existing user account will be able to easily log in and access all functionalities, for new users, the application enables quick and simple registration, which will enable the purchase of products and their delivery.

It is important to emphasize that with our web application, users are not required to log in to the site to view the products that are available. It's easy to use the app as a guest, but in this way, users don't have the option to buy the products. This functionality gives users the freedom to test the application before deciding to create their own account that allows the use of all the advanced functionalities that the application offers.
